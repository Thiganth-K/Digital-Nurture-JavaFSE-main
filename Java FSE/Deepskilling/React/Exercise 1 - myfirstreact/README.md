# Exercise 1 - My First React App (HOL 1)

## Topic
Setting up a React environment and creating first React application.

## Objective
- Define SPA and its benefits
- Understand React and virtual DOM
- Set up React environment using `create-react-app`

## Task
Create a new React Application with the name "myfirstreact" and run the application to print:
> **"Welcome to the first session of React"** as a heading on the page.

## How to Run
```bash
cd "Exercise 1 - myfirstreact"
npm install
npm start
```
Then open browser at: http://localhost:3000

## Expected Output
A dark-themed page displaying:
```
Welcome to the first session of React
```
as an `<h1>` heading.

## Files
- `src/App.js` — Main component rendering the heading
- `src/index.js` — Entry point rendering App into DOM
- `public/index.html` — HTML template
